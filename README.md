# API-Automation
This project demonstrates automated API testing using Python. It validates endpoints, ensures data
integrity, and detects issues using structured test cases.
## Features:
- Automated API testing with pytest
- HTTP requests using requests
- JSON Schema validation
- Data-driven testing (JSON/CSV)
- Reusable API client

## Tools & Technologies
Python, pytest, requests, jsonschema
Project Structure
tests/ - API test cases
utils/ - API client & data loader
schemas/ - JSON validation
data/ - test data
conftest.py - fixtures

## How to Run
1. pip install -r requirements.txt
2. pytest

Showcases skills in API testing, automation, and Python scripting.
